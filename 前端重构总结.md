## 二、功能流程变更

### 旧流程（爬虫模式）
```
启动页 → 启动爬虫 → 自动采集数据 → 仪表盘显示 → AI分析 → 下注
```

### 新流程（手动模式）
```
启动页 → 选择桌号 → 上传页面 → 输入开奖记录 → 仪表盘显示五路图 → AI分析 → 下注 → 等待开奖 → 输入结果 → 结算 → 下一局
```
