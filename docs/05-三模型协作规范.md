# 三模型协作规范

## 角色职责
- 庄模型：只输出庄向证据链
- 闲模型：只输出闲向证据链
- 综合模型：融合证据与走势图输出最终预测与置信度
- 三个模型均不分析和局，综合模型只对下一局庄或闲做预测

## 协作方式
- 庄/闲并行
- 综合模型串行汇总

## 输出结构约定（用于前端智能分析板块）
- 庄模型输出：庄向证据摘要、关键信号、信号强度、反向风险点。
- 闲模型输出：闲向证据摘要、关键信号、信号强度、反向风险点。
- 综合模型输出：融合摘要、冲突处理说明、最终结论（下局庄/闲）、结论理由、当前置信度。
- 可读约束：面向小白展示“可读摘要”，不输出冗长原始推理文本。
- 一致性约束：综合模型最终结论必须与顶部“预测局庄/闲”一致，出现不一致时按异常处理写入实盘日志。

## 五路因子摘要模板
- 庄模型模板：`因为大路{要点}、珠盘路{要点}、大眼仔路{要点}、小路{要点}、螳螂路{要点}，所以{庄向结论}`。
- 闲模型模板：`因为大路{要点}、珠盘路{要点}、大眼仔路{要点}、小路{要点}、螳螂路{要点}，所以{闲向结论}`。
- 综合模型模板：`因为{庄闲证据对比结论}+{五路冲突处理结果}，所以最终结论下局预测{庄/闲}`。
- 文案长度：每个“要点”建议6到12字，整句建议30到80字，必须简洁易懂。

## 动态学习与版本协同
- 综合模型支持AI学习后版本迭代，形成可追踪模型版本号。
- 启动系统后由AI记忆模块自动选择最佳模型版本参与本轮分析预测。
- 预测策略允许随学习与复盘动态调整，保持灵活多变而非固定死规则。

## 连续失准应对机制
- 当综合模型连续3局预测错误时，立即触发策略重评估动作。
- 策略重评估内容包括：最近局特征重加权、错题本本靴复盘强化、置信度阈值收紧。
- 连续失准期间仍保持每局必下注，但可切换为保守下注参数并强化复盘，保证验证连续性。
- 连续失准触发仅属于在线重评估，不等同于离线AI学习任务。
