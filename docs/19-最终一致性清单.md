# 最终一致性清单

## 文档目标
- 用一页检查系统是否存在逻辑冲突、规则遗漏与实现偏移。
- 本清单用于上线门禁勾选，不替代14号文档的测试执行步骤与量化细则。

## 规则一致性
- [ ] 仅预测庄/闲，不预测和局
- [ ] 每个有效下注局必下注，下注为仿真验证
- [ ] 每个有效结算局必须进入已结算或异常退回
- [ ] 下注金额智能自适应，满足最小10、最大10000、10的倍数
- [ ] 连续3错后只能降档，不可停注

## 靴与局一致性
- [ ] 新靴判定使用局号递减双确认
- [ ] 错题本严格本靴内生效，不跨靴
- [ ] 本靴进度条与实际局号一致

## 注单与结算一致性
- [ ] 每个有效下注局存在且仅存在一条下注记录
- [ ] 每条下注最终进入已结算或异常退回二选一
- [ ] 异常退回幂等，不存在重复退回
- [ ] 5分钟退回统一以服务器入库下注时间为基准
- [ ] 数据异常仅作为中间诊断态，不可作为资金终态

## 模型与学习一致性
- [ ] AI学习仅在数据库历史有效局总量达到200局后可启动
- [ ] 学习中有任务锁，禁止重复触发
- [ ] 学习完成有预计时间对照与学习总结
- [ ] 自动选模遵守冷却期，避免频繁切换
- [ ] 连续3错仅触发在线重评估，不自动拉起离线AI学习
- [ ] AI学习按靴执行，单次学习不跨靴混样

## 前端与数据一致性
- [ ] 智能分析字段与模板实现与20号文档一致
- [ ] 当前靴5路图与目标网站布局一致，仅叠加错误标记层
- [ ] 同一错误在5路图、开奖记录、下注记录中使用同一错误编号
- [ ] 实盘日志、下注记录、开奖记录三者时间线一致
- [ ] 启动页选桌点击启动后，进入首页同时自动触发智能选模（有可用版本时）
- [ ] 首页智能分析板块展示庄模型、闲模型、综合模型三类摘要
- [ ] 庄模型与闲模型摘要均包含五路要点（大路、珠盘路、大眼仔路、小路、螳螂路）
- [ ] 综合模型“因为…所以…”结论与顶部预测方向一致
- [ ] 实盘日志采用首行字段与折叠字段双层展示，字段口径一致
- [ ] 日志事件分类与优先级映射一致，超时告警归属P1并置顶
- [ ] 系统健康分可解释且可追溯到子项指标，并满足阈值口径一致（>=85正常，70-84观察，<70告警）

## 安全与审计一致性
- [ ] 管理员首次登录必须修改默认密码
- [ ] 连续输错密码5次锁定10分钟
- [ ] 锁定与解锁事件写入实盘日志并可检索

## 验收证据绑定
- [ ] 每条验收项均可绑定至少一种证据（日志片段、界面截图、记录条目、导出复盘包）

## 导出与复盘一致性
- [ ] 一键导出复盘包包含本靴日志、5路图、错题、版本号、流水
- [ ] 复盘包可用于历史回放验证版本有效性
