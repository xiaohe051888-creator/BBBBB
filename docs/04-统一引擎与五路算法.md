## 🎯 标准规则总结

### 1. **大路（Big Road）**

#### 显示样式
- **庄(Banker)**：红色实心圆圈 ✅
- **闲(Player)**：蓝色实心圆圈 ✅
- **和(Tie)**：绿色实心圆圈（显示在上一局位置）✅

#### 布局规则
1. **相同结果往下排**（纵向延伸）
2. **不同结果换新列**（从下一列第一行开始）
3. **每列最多6个点**，超过则在第6个位置向右平行延伸 ✅
4. **和局不换列**：在上一局对应位置显示绿色实心圆
   - 不换列，不换行，不开启新列
   - 如果上一局是和局，则继续在同一位置叠加

#### 颜色分组
- 连续相同颜色的结果为一组
- 颜色变化必换列

### 2. **珠盘路（Bead Road/Plate）**

#### 标准网格
- **固定尺寸**：14列 × 6行 ✅
- **容量**：最多84个结果（14×6）
- **布局顺序**：从上到下、从左到右填充网格

#### 显示内容
- **庄**：红色实心圆，里面写"庄"字 ✅
- **闲**：蓝色实心圆，里面写"闲"字 ✅
- **和**：绿色实心圆，里面写"和"字 ✅

#### 填充规则
1. 先填满第一列的所有行（从上到下）
2. 再填第二列的所有行
3. 超过84个点时，旧位置被新数据覆盖（取模行为）

### 3. **下三路（Derived Roads）**

#### **⚠️ 重要区别**：颜色意义完全不同
- **红色**：代表"有规律"（Predictability/Consistency）✅
- **蓝色**：代表"随机/混沌"（Chaos/Randomness）✅
- **颜色不代表庄闲**，只代表模式一致性 ✅

#### 显示样式
| 路名 | 英文名 | 显示样式 | 说明 |
|------|--------|----------|------|
| 大眼仔路 | Big Eye Boy | **实心圆** | 比大路的圆略小 |
| 小路 | Small Road | **空心圆** | 实心点（BaccaratTraining.com标准） |
| 蟑螂路 | Cockroach Pig | **斜杠（/）** | 红色/蓝色斜线 |

#### 布局规则
- **与大路完全相同的布局规则**：
  - 相同结果往下排（纵向延伸）
  - 不同结果换新列
  - 每列最多6个点，超过向右平行延伸 ✅

#### 启动条件
| 路名 | 开始条件 | 比较偏移量(k) |
|------|---------|--------------|
| 大眼仔路 | 大路第2列出现第一个结果后 | k=1 |
| 小路 | 大路第3列出现第一个结果后 | k=2 |
| 蟑螂路 | 大路第4列出现第一个结果后 | k=3 |

### 4. **核心算法：两种情形判断法**

#### **参数定义**
- **k**：比较偏移量（大眼仔=1, 小路=2, 蟑螂路=3）
- **m**：最新结果在大路中的行号（1-based，从上到下）
- **n**：最新结果在大路中的列号（1-based，从左到右）

#### **情形A：新列出现（m=1，大路开启新列）**
- **比较**：新列左边第1列 vs 左边第(k+1)列的**高度是否相等**
- **结果**：
  - 高度相等 → 红(延) → 有规律
  - 高度不等 → 蓝(转) → 随机/断裂

#### **情形B：延续当前列（m≥2，大路在当前列向下添加）**
- **检查**：从当前点位置向左移k格，再向上移1格
- **判断**：该位置**是否有值**
- **结果**：
  - 有值（同行有数据）→ 红(延) → 规律延续
  - 无值（同行无数据）→ 蓝(转) → 规律断裂

## 🔧 统一引擎实现

### 后端引擎 (`road_engine.py`)

```python
class UnifiedRoadEngine:
    """统一路引擎 - 五路走势图计算（权威标准版）"""
    
    # === 标准规则常量 ===
    MAX_ROWS_PER_COLUMN = 6   # 大路/派生路每列最多6个点（全球统一）
    BEAD_COLUMNS = 14         # 珠盘路固定14列
    BEAD_MAX_ROWS = 6         # 珠盘路固定6行
    
    def __init__(self):
        self.current_game_numbers: List[int] = []  # 当靴所有局号
        self.current_results: List[str] = []       # 当靴所有结果（庄/闲/和）
        self.error_map: Dict[int, str] = {}        # 局号 -> 错误编号
```
