# 核心指标与验收

## 核心指标
- 爬虫采集准确率
- 五路算法规则正确性
- 三模型预测准确率
- 前端用户体验
- AI学习有效性（学习后准确率提升幅度）
- 启动智能选模效果（所选版本实盘命中率）
- 连续失准恢复能力（连续3次错误后的恢复效果）
- 必下注覆盖率（每局下注执行率应为100%）
- 自适应下注有效性（金额调整与命中率/回撤的匹配效果）
- 系统健康分（采集稳定性+模型稳定性+结算一致性）

## 验收口径
- 明确每项指标的统计周期、统计口径与达标阈值。
- 模型版本最多保留5个，超出后按策略淘汰旧版本并保留可追溯记录。
- 准确率统计口径仅包含庄/闲预测，不包含和局预测。
- 本系统下注为仿真验证行为，不与目标网站产生真实下注交互。
- 自适应下注验收要求：下注金额每局可追溯、边界合法、无越界记录。
- 连续3错应对口径：只能降档，不可停注。
- 连续3错触发口径：仅触发在线重评估，不自动触发离线AI学习。
- 系统健康分阈值口径：`>=85`正常，`70-84`观察，`<70`告警。
