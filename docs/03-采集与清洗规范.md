# 采集与清洗规范

## 采集规则
- 每10秒轻量探测局号
- 局号相同不重复抓取完整数据
- 局号递增触发完整抓取
- 无换桌情况下，局号递减判定为换靴候选事件
- 连续10分钟未获取到有效数据时，进入洗牌等待态；每10分钟继续探测一次

## 清洗规则
- 字段完整性检查
- 异常数据标注与重试策略

## 新靴判定规则
- 满足“局号递减 + 连续两次探测确认”才正式判定新靴开始
- 新靴开始后重置本靴统计、错题本、进度条与本靴健康分子项
- 若仅出现一次局号递减且后续恢复递增，按采集抖动处理，不触发换靴

## 有效局定义（两阶段口径）
- 有效下注局：检测到新局号，且下注所需字段完整可用，记为一个有效下注局。
- 有效结算局：有效下注局对应注单最终进入“已结算”或“异常退回”，记为一个有效结算局。
- 数据异常口径：数据异常仅为中间诊断态，不计入有效结算终态，必须继续流转到“已结算”或“异常退回”。
- 无效局：重复局号、字段缺失且重试失败、采集抖动回滚局号，均不进入下注与结算统计。
- 记录约束：每个有效下注局必须产生一次仿真下注记录；每个有效结算局必须产生一次结算或异常退回记录。
