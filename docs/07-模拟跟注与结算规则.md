# 模拟跟注与结算规则

## 默认参数
- 默认资金：20000
- 庄赔率：1.95
- 闲赔率：2.0
- 和局不输钱
- 最小下注：10
- 最大下注：10000
- 下注步长：10的倍数
- 下注金额模式：智能自适应（按最近表现、置信度、回撤状态动态调整）

## 关键规则
- 下注即扣款
- 开奖即结算
- 下注超5分钟未开奖：异常注单自动退回（以服务器入库下注时间为计时起点）
- AI预测与下注仅覆盖庄/闲，不对和局做预测与下注
- 若开奖结果为和局，按和局不输钱规则退回本金并完成结算
- 每个有效下注局必须下注，不允许跳过有效下注局
- 本系统下注为仿真验证数字，不会向目标开奖网站发起真实下注行为
- 自适应后下注金额仍需满足最小10、最大10000、10的倍数约束

## 自适应下注金额规则
- 输入因子：当前置信度、最近连续正确/错误、当前回撤水平、模型版本稳定性评分
- 调整方向：表现好时缓升、连续失准时降档，始终保持每局下注
- 风控边界：任意情况下不突破最小/最大/步长限制
- 可追溯要求：每局记录下注金额计算依据摘要并写入实盘日志

## 三档自适应规则表
- 保守档：连续3次预测错误或当前回撤偏高时使用，下注系数0.5
- 标准档：常规状态使用，下注系数1.0
- 进取档：连续正确且置信度高时使用，下注系数1.5

## 金额计算与取整
- 基础金额：`base_amount = 100`
- 计算公式：`raw_amount = base_amount × 档位系数`
- 取整规则：`bet_amount = floor(raw_amount / 10) × 10`
- 边界裁剪：`bet_amount = min(10000, max(10, bet_amount))`
- 最终要求：每个有效下注局都按上述规则生成下注金额，并写入下注记录与实盘日志

## 异常退回幂等规则
- 触发条件：下注后超过5分钟未开奖，或系统停止、换桌、系统故障导致无法完成结算
- 时间基准：统一使用服务器入库下注时间计算5分钟窗口
- 幂等约束：同一注单最多执行一次异常退回
- 互斥约束：注单进入异常退回后，不可再进入已结算状态
- 诊断态约束：数据异常仅作为中间诊断标记，不可替代已结算/异常退回资金终态。
